# Chebyshev–Sylvester method, expository + computational review (Gantumur 2025)

Detailed expository and computational account of the elementary Chebyshev/Sylvester
bounds on π(x): the method of "replacing the Möbius function with a finitely supported
arithmetic function in the convolution identities" — i.e., exactly P7's move. Modern
(Dec 2025), with computational verification of the historical results.
