# Posets, Tensor Products and Schur positivity

**Authors:** Vyjayanthi Chari, G. Fourier, Daisuke Sagaki  ·  **Year:** 2012  ·  **Source:** https://arxiv.org/abs/1210.6184

## One-line
Defines a partial order on $k$-tuples of dominant weights summing to a fixed $\lambda$, and shows tensor-product dimensions (and in key cases, characters) increase Schur-positively along the order.

## Key claim
For a simple Lie algebra $\mathfrak{g}$ and $\lambda \in P^+(\lambda,k)$, if $\boldsymbol\lambda \preceq \boldsymbol\mu$ in $P^+(\lambda,k)/\!\sim$ then $\dim \bigotimes_s V(\lambda_s) \le \dim \bigotimes_s V(\mu_s)$ (equality iff equivalent). For $\lambda = N\omega_i$ with $\omega_i$ minuscule, and for $\mathfrak{g}=\mathfrak{sl}_3$ with $k=2$, the inequality lifts to a Hom-multiplicity (Schur-positivity) inequality.

## Method
Reduce dimension monotonicity to $\mathfrak{sl}_2$ via the Weyl dimension formula by showing the partial order on $P^+(\lambda,k)$ is generated by the cover relations of the $k=2$ case ($\lambda_{j_2}, \lambda_{j_1}) \to (\lambda_{j_2}-1, \lambda_{j_1}+1)$). For minuscule $\omega_i$, construct an explicit injection between Littelmann LS-path sets $B(r\omega_i)^{\mu\text{-dom}}_{s\omega_i} \hookrightarrow B((r{+}1)\omega_i)^{\mu\text{-dom}}_{(s{-}1)\omega_i}$ by rescaling rationals $a \mapsto ra/(r{+}1)$ and appending $(r{+}1)\omega_i$. For $A_2$, $k=2$, use Kashiwara–Nakashima semistandard tableau parameterizations $T(\lambda)^\nu_\mu \subset \mathbb{Z}_+^5$ and exhibit explicit injective reparameterizations of the 5-tuples $(s_{1,1},s_{1,2},s_{1,3},s_{2,2},s_{2,3})$.

## Result
Theorem 1: (i) dimension monotonicity along $\preceq$ for all $\mathfrak{g},\lambda,k$; (ii) Hom-multiplicity monotonicity (hence Schur-positivity in type $A$) when $\lambda = N\omega_i$ for $\omega_i$ minuscule; (iii) the same for $\mathfrak{g} = A_2$, $k=2$, all $\lambda$. For type $A_n$, $k=2$, the row-shuffle of [FFLP05] is identified as the unique maximal element $\lambda^{\max}$ of $P^+(\lambda,2)/\!\sim$; on $\mathfrak{sl}_2$, equivalence classes coincide with $S_k$-orbits. Combinatorial corollary: if $\sum_{j\le i} a_j \le \sum_{j\le i} b_j$ for all $i$ then $\prod a_j \le \prod b_j$ (equality iff equal).

## Why it matters here
General background; no direct arena tie. Sits in symmetric-function / representation-theory combinatorics (Schur positivity, Littlewood–Richardson) — closest arena adjacency would be extremal-combinatorics framings, but no direct optimization-bound application.

## Open questions / connections
- Conjecture (Lam–Postnikov–Pylyavskyy, restated here): the Hom inequality (iii) holds for all simple $\mathfrak{g}$, all $\lambda$, $k=2$ — only $A_2$ and minuscule cases are proved.
- Determining cover relations of $P^+(\lambda,2)/\!\sim$ for general $\mathfrak{g}$ beyond $A_2,A_n$; the authors note dependence on root-system combinatorics.
- Extends [LPP07]'s row-shuffle Schur positivity (maximal-element case in type $A$) to all comparable pairs; [DP07] gives support-containment evidence via Horn–Klyachko in type $A$.
